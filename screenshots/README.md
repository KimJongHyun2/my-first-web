스크린샷 요약

이 폴더에는 Ch7 검증용으로 캡처한 화면들의 메타정보가 들어 있습니다. 실제 이미지 파일은 채팅 UI에서 생성된 캡처를 참고하세요.

파일명 규칙 (권장)
- home-desktop.png        — 홈 페이지 (데스크탑)
- posts-desktop.png       — 게시글 목록 (데스크탑)
- newpost-desktop.png     — 새 글 쓰기 (데스크탑)
- home-tablet.png         — 홈 페이지 (태블릿 768×1024)
- posts-mobile.png       — 게시글 목록 (모바일 375×812)
- newpost-mobile.png     — 새 글 쓰기 (모바일 375×812)

포함된 캡처(요약)
- 데스크탑: 홈 / 게시글 목록 / 새 글 쓰기
- 태블릿: 홈(768×1024)
- 모바일: 게시글 목록 / 새 글 쓰기(375×812)

추가 안내
- 제출용으로 실제 PNG 파일이 필요하시면 로컬에서 `http://localhost:3000`을 띄운 상태에서 브라우저 개발자 도구 또는 `puppeteer`/`playwright` 스크립트를 사용해 저장하세요.
- 원하시면 제가 자동 스크립트(Playwright)를 만들어 repo에 추가해 드릴게요.
